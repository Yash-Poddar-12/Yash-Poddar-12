## Featured Projects

### 1. [Reinforcement Learning Autoscaler](https://github.com/Yash-Poddar-12/serverless_edge_computing)
Built a deep reinforcement learning based autoscaling system for serverless workloads using Stable-Baselines3 and a custom simulation environment.

### 2. [Spectral Denoising with Deep Learning](https://github.com/Yash-Poddar-12/spectral_denoise)
Developed a PyTorch-based 1D U-Net for spectral denoising with a reproducible training and evaluation pipeline.

### 3. [AI-Based OS Task Scheduler](https://github.com/Yash-Poddar-12/Stigmergic_Sentinels)
Designed a multi-pheromone swarm intelligence scheduler inside a custom discrete-event OS simulation and benchmarked it against baselines.

### 4. Telemetry and Ground Control Systems
Built real-time telemetry, logging, replay, and remote command workflows for rocket testing and ground control.
